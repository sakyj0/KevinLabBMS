## 기술적 결정사항
- **프론트엔드 우선 개발 방식** 채택: UI와 사용자 경험을 먼저 구현하고 추후 Firebase 백엔드 연동
- **바닐라 JavaScript** 사용: 프레임워크 의존성 없이 순수 JavaScript로 구현하여 유지보수성 향상
- **모듈화된 컴포넌트 구조**: 재사용 가능한 컴포넌트로 분리하여 확장성 확보
- **SOLID 원칙 적용**: 단일 책임 원칙을 중심으로 클래스 설계
